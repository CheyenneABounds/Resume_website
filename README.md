# Resume Website

A lightweight, static personal resume website. Built with plain HTML, CSS, and minimal JavaScript for interactivity. The site showcases professional experience, skills, education, and contact information.

Contents
- `index.html` — Main portfolio / resume page with hero, about, experience, skills, and connect CTA.
- `contact.html` — Contact page with contact form and contact methods.
- `css/style.css` — Central stylesheet (design tokens, components, responsive rules).
- `js/main.js` and `js/contact.js` — Minimal JS for navigation and form handling.

Notes
- Accessibility improvements were applied: ARIA labels, focus-visible styles, and improved hover/focus contrast on buttons.
- External links include `rel="noopener noreferrer"` for security.

Contributing
- Make edits locally and open a PR. Keep changes scoped and preserve design tokens in `css/style.css`.
